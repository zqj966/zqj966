# Hi, I’m a learner in Shenzhen 👋

I’m currently studying and building small research tools around **multimodal large models** and **speech foundation models**. My recent focus is on evaluation, data pipelines, and the less glamorous parts of model research that make experiments reproducible.

## Current interests

- Multimodal dialogue evaluation and grounded reasoning
- Speech tokenization, acoustic representations, and lightweight audio experiments
- Clean research code: small datasets, reproducible configs, readable baselines
- Practical ML tooling with Python, PyTorch, NumPy, and Hugging Face-style workflows

## Featured local projects

| Project | Direction | Description |
